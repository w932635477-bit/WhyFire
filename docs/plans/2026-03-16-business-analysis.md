## 附录：成本计算公式

### AI 成本公式

```
单次分析成本 = (输入tokens × 输入价格 + 输出tokens × 输出价格) / 1,000,000

示例（Haiku 竞品分析）：
= (2000 × $1 + 1500 × $5) / 1,000,000
= ($2,000 + $7,500) / 1,000,000
= $0.0095
≈ ¥0.07
```

### 毛利率公式

```
毛利率 = (收入 - AI成本 - 支付手续费) / 收入 × 100%

示例（训练版）：
= (¥29 - ¥12.50 - ¥0.16) / ¥29 × 100%
= 56%
```

### 盈亏平衡公式

```
盈亏平衡用户数 = 固定成本 / 平均单用户毛利
= ¥280 / ¥17
= 17 用户/月
```

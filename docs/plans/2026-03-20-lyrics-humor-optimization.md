# 歌词生成幽默优化设计文档

## 背景与目标

### 问题
当前歌词生成 Prompt 过于泛泛，只要求"接地气"、"押韵自然"，生成的歌词"有点水"，缺乏幽默感和记忆点。

### 目标
优化歌词生成 Prompt，让 AI 掌握制造幽默感的技巧，生成更搞笑、更有梗、更接地气的歌词。

### 核心定位
WhyFire 是利用 Rap 视频帮用户做推广的工具，歌词质量直接影响传播效果和用户留存。

---

## 设计方案

### 1. 搞笑技巧库

在 Prompt 系统中内置完整的搞笑技巧库，包含 7 种核心技巧：

| 技巧 | 原理 | 示例 |
|------|------|------|
| **夸张法** | 把特点放大到荒诞程度 | "这耳机降噪强到我妈喊我吃饭我都以为在拍恐怖片" |
| **反差/反转** | 预期 vs 现实的落差感 | "老板说要给我涨薪...涨了50块，够买瓶可乐" |
| **谐音梗** | 读音相似制造记忆点 | "这产品很'牛'，不是英文的'new'，是真的牛" |
| **神转折** | 正经铺垫+沙雕收尾 | "产品质量过硬，客服态度超好，唯一缺点是...我买不起" |
| **自嘲/吐槽** | 自黑增加亲切感 | "虽然钱包很薄，但这不妨碍我眼光很毒" |
| **拟人化** | 让产品"说话" | "我的电池从来不加班，所以续航特别长" |
| **网络热梗** | 融入当下流行语 | "这波操作，属于是狠狠拿捏了" |

### 2. 场景-技巧映射

根据不同创作场景，智能推荐技巧组合：

| 场景 | 推荐技巧 | 理由 |
|------|---------|------|
| **产品推广** | 夸张 + 谐音梗 + 神转折 + 拟人化 | 突出卖点同时保持趣味，不让广告感太重 |
| **搞笑洗脑** | 反转 + 吐槽 + 夸张 + 热梗 | 最大化娱乐效果，追求病毒传播 |
| **IP混剪** | 夸张 + 热梗 + 反转 | 符合粉丝文化，增加共鸣 |
| **日常Vlog** | 自嘲 + 吐槽 + 反转 | 真实接地气，引发共情 |

### 3. Few-Shot 示例

为每种场景提供 1-2 个优质歌词示例，让 AI 学习"好歌词长什么样"。

**产品推广示例（蓝牙耳机）：**
```
降噪强到邻居以为我在拍恐怖片
续航长得像我的单身年限
价格便宜到我妈都问是不是假的
别问，问就是真香定律
```

**搞笑洗脑示例（加班主题）：**
```
老板说要给我涨薪 我激动得差点当场起飞
结果涨了五十块 够买瓶可乐压压惊
加班到凌晨 朋友圈只有外卖小哥点赞
这波属于是狠狠地破防了
```

### 4. 质量要求

优化后的 Prompt 需要满足：
- **至少使用 2-3 种搞笑技巧**
- **有 1-2 个"金句"（可单独传播的句子）**
- **押韵自然，节奏感强**
- **控制字数：100-150字**

## 技术实现

### 修改文件
1. `src/lib/ai/prompts/lyrics-prompts.ts` - 主 Prompt 文件
2. `src/lib/ai/prompts/humor-techniques.ts` - 新建搞笑技巧库（可选，也可以直接整合到 lyrics-prompts.ts）

### 实现方式
1. 创建 `HUMOR_TECHNIQUES` 常量，包含技巧定义和示例
2. 创建 `SCENE_HUMOR_MAP` 映射表
3. 创建 `LYRICS_EXAMPLES` few-shot 示例库
4. 修改 `buildProductPrompt`、`buildFunnyPrompt` 等函数，整合幽默技巧指导
5. 添加方言版本的幽默技巧（东北话、粤语等）

---

## 验收标准

1. 生成的歌词明显更有幽默感
2. 歌词中能识别出 2-3 种搞笑技巧
3. 有可以单独传播的"金句"
4. 不同场景的歌词风格有明显区分
5. 方言版本保持幽默感（东北话、粤语）

## 钩子模板结构

每个钩子模板包含以下信息：

```json
{
  "hook_id": "唯一ID",
  "type": "钩子类型",
  "script_cn": "中文文案模板",
  "script_en": "英文文案模板",
  "example": "具体示例",
  "source": "来源平台",
  "tags": ["标签1", "标签2"],
  "suitable_for": "适用场景",
  "tips": ["使用技巧"]
}
```

## 钩子模板使用指南

### 1. 选择合适的钩子类型

| 内容类型 | 推荐钩子类型 |
|---------|-------------|
| 教程/干货 | 好奇型、价值承诺型、数字型 |
| 评测/体验 | 反差型、好奇型 |
| 故事/Vlog | 故事型、个人经历型 |
| 知识科普 | 提问型、好奇型 |
| 产品推荐 | 紧迫型、价值承诺型 |

### 2. 钩子公式

```
钩子 = 好奇/痛点 + 情绪 + 承诺

示例：
- 好奇："你有没有想过..."
- 痛点："你是不是也..."
- 情绪："我真的没想到..."
- 承诺："这3个方法帮你..."
```

### 3. 黄金3秒法则

| 时间 | 要素 |
|------|------|
| 0-1秒 | 视觉冲击/表情/动作 |
| 1-2秒 | 第一句话（钩子） |
| 2-3秒 | 强化钩子/引出主题 |

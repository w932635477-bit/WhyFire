## 📋 使用方法

### 方法一: 通过 Admin 页面生成

1. 访问 `http://localhost:3002/admin/background`
2. 复制上面的 Prompt
3. 点击 "Generate Background"
4. 等待 30-60 秒生成完成
5. 下载图片

### 方法二: 转换为视频

由于 Evolink 生成的是静态图片，你可以：

1. **使用视频编辑软件**
   - 将生成的图片导入 Premiere/Final Cut
   - 添加缓慢的缩放或平移效果（Ken Burns 效果）
   - 添加轻微的动态模糊
   - 导出为 15-30 秒循环视频

2. **使用 AI 视频工具**
   - Runway Gen-2
   - Pika Labs
   - Sora (如有访问权限)
   - 将静态图片转为动态视频

3. **使用在线工具**
   - Canva Video
   - Kapwing
   - 添加动态效果和过渡

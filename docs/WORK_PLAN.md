## 六、风险管理（新方向）

| 风险 | 应对措施 |
|------|---------|
| TikTok API 不稳定 | 使用 Apify 作为备选方案 |
| 视频下载被限制 | yt-dlp 多种下载方式 |
| Claude API 成本高 | 缓存分析结果，批量处理 |
| 本土化建议质量差 | Prompt 迭代优化，A/B 测试 |
| 海外内容不适合国内 | 用户反馈筛选，优先选择可借鉴内容 |

### 技术风险（v2.0 优化版）

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|---------|
| YouTube API 配额用尽 | 低 | 高 | 缓存热门数据，优化请求 |
| Apify 服务不稳定 | 低 | 中 | YouTube 为主，Apify 为辅 |
| AI 分析质量不稳定 | 中 | 中 | Prompt 迭代优化 |
| 本土化建议不适合 | 中 | 中 | 用户反馈迭代 |

## 十、技术栈确认（v2.0 优化版 - 2026-03-17）

### 前端
- Next.js 15（App Router）
- React 18
- Tailwind CSS
- 瀑布流组件（CSS Grid / Masonry）

### 后端
- Next.js API Routes（单一技术栈）

### AI 服务
- Claude Haiku（文本分析，$0.25/1M tokens）

### 数据存储
- SQLite（MVP 阶段）

### 数据源
- YouTube Data API v3（主数据源，官方免费）
- Apify TikTok Scraper（辅助数据源，$49/月）

### 成本估算
```
YouTube API: 免费
Apify: $49/月
Claude Haiku: ~$5-10/月
总计: ~$55/月 ≈ ¥400/月
```

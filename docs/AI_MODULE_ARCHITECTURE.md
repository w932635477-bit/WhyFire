## 成本估算

```
┌─────────────────────────────────────────────────────────────────┐
│                      API 成本估算（每次分析）                     │
└─────────────────────────────────────────────────────────────────┘

Claude Sonnet 4 (竞品分析 + 内容诊断)
├─ Input:  ~1000 tokens × $3/1M = $0.003
├─ Output: ~1500 tokens × $15/1M = $0.0225
└─ Total:  $0.0255 per analysis

Claude Haiku 3.5 (灵感解读)
├─ Input:  ~500 tokens × $0.25/1M = $0.000125
├─ Output: ~800 tokens × $1.25/1M = $0.001
└─ Total:  $0.001125 per analysis

完整流程成本:
├─ Step 1: 竞品分析 (Sonnet) = $0.0255
├─ Step 2: 内容诊断 (Sonnet) = $0.0255
├─ Step 4: 灵感解读 × 8 (Haiku) = $0.009
└─ Total: ~$0.06 per user session

月度成本（1000 用户）:
1000 users × $0.06 = $60/month
```

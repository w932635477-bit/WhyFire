|-----------|-----------|
| 提升留存率只是表面问题 | 解决"不知道拍什么"的根本痛点 |

| 需要复杂分析框架 | AI自动解读更直接高效 |

## 六、商业协同

### WhyFire + VidLuxe 产品矩阵

```
WhyFire（诊断+建议）→→→ VidLuxe（工具执行）
        ↑                       ↓
        ←←← 用户口碑传播 ←←←←←←←←
```

### 协同逻辑

| 产品 | 定位 | 价值 |
|------|------|------|
| **WhyFire** | AI 教练（诊断） | 告诉你问题在哪，你能变得多好 |
| **VidLuxe** | 工具箱（执行） | 一键帮你优化封面、图片 |

### 软性引导策略

- **不是强制**：给用户建议，而非强制执行
- **展示对比**：封面前后对比图，让用户自己意识到差距
- **自然导流**：用户想优化时，VidLuxe 是最方便的选择

### 核心依赖（精简版）

| 依赖 | 用途 | 成本 |
|------|------|------|
| **YouTube Data API v3** | YouTube 热门数据 | 免费（10,000单位/天） |
| **Apify TikTok Scraper** | TikTok 热门数据 | $49/月（可按需） |
| **Claude Haiku** | 文本分析 | ~$5-10/月 |
| **Next.js 15** | 全栈框架 | 开源 |

**月成本估算：约 ¥400/月**

### 为什么选择文本分析而非视频分析

| 维度 | 视频分析 | 文本分析 |
|------|---------|---------|
| 成本 | $3/1M tokens | $0.25/1M tokens |
| 延迟 | 数秒 | 毫秒级 |
| 信息量 | 视觉细节 | 元数据足够 |
| 复杂度 | 需要 FFmpeg | 纯 API |
| 适用场景 | 需要理解画面 | 理解创意概念 |

**结论**：对于"创意灵感"场景，标题/描述/标签已包含核心创意信息

## 七、信任建立策略

### 信任金字塔

```
第一层：零门槛体验
- 完全免费试用，无需注册
- 粘贴链接 → 立即看到结果
- 先给价值，再谈付费

第二层：真实案例展示
- 展示真实爆款的分析报告
- "这条笔记为什么有 10 万赞"
- 用真实数据证明 AI 分析的价值

第三层：透明度
- 告诉用户 AI 如何分析（不是黑盒）
- 展示分析依据："标题得分 85 分，因为..."
- 让用户能验证结果的合理性

第四层：隐私承诺
- 明确声明：不存储用户笔记内容
- 只分析公开信息，不触碰私密
- 可选：提供"分析后立即删除"选项

第五层：社会认同
- 用户评价/反馈展示
- "xxx 博主说有用"
- 累计分析 xx 条笔记
```

## 十一、技术决策记录

### 2026-03-17：方向调整（重大）

**决策**：从"爆款开头分析"转向"参照系扩宽"

**理由**：
1. 用户反馈"不知道拍什么"比"开头不够好"更严重
2. 海外创意灵感库是更稀缺的价值
3. 提升创作灵感比优化单点技巧更有意义
4. 多模态 AI 视频理解技术已成熟

**旧方案（放弃）**：
- @lucasygu/redbook 爆款分析
- Playwright 小红书抓取
- 透明化分析框架

**新方案（采用）**：
- TikTok/IG/YouTube 趋势抓取
- 多模态 LLM 视频理解
- 本土化建议生成

---

### 2026-03-17：技术选型 v2.0（优化版）

**数据抓取**：
| 平台 | 方案 | 理由 |
|------|------|------|
| YouTube（主） | YouTube Data API v3 | 官方免费稳定，70%数据来源 |
| TikTok（辅） | Apify TikTok Scraper | 商业服务稳定，30%数据来源 |

**AI 分析**：
| 功能 | 方案 | 理由 |
|------|------|------|
| 文本分析 | Claude Haiku | 极低成本 $0.25/1M tokens |
| 本土化建议 | Claude Haiku | 与文本分析统一 |

**放弃的技术**：
- ~~Playwright + 小红书抓取~~（旧方向）
- ~~Claude Vision 视频分析~~（成本高）
- ~~TikTok-Api Python~~（非官方 API 不稳定）
- ~~FFmpeg 关键帧提取~~（不做视频处理）

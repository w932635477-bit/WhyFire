# WhyFire 用户流程图

> 版本：v2.1
> 最后更新：2026-03-17
> 设计原则：向导式流程，一环扣一环

---

## 一、产品定位

**WhyFire = 小红书 AI 教练**

核心价值：帮助 0-1万粉起步博主找到差距，获得改进建议

---

## 二、四大功能模块

| 模块 | 核心价值 | 数据源 |
|------|---------|--------|
| **竞品分析** | 分析爆款笔记，拆解成功要素 | 小红书公开数据 |
| **内容诊断** | 分析自己内容，找出差距 | 小红书公开数据 |
| **可视化对比** | 对比差距，一目了然 | 综合前两步数据 |
| **海外创意灵感库** | 精准推荐 6-10 个相关海外爆款 | YouTube/TikTok API |

## 五、关键设计决策

### 5.1 为什么是向导式流程？

| 原因 | 说明 |
|------|------|
| **流程连贯** | 每一步依赖前一步的数据 |
| **降低认知负担** | 用户不需要思考"该做什么" |
| **提升完成率** | 引导用户走完全程 |
| **教练感** | AI 教练一步步引导，不是冷冰冰的工具 |

### 5.2 为什么海外灵感库是精准推荐？

| 原因 | 说明 |
|------|------|
| **避免信息过载** | 海量视频让用户无所适从 |
| **珍惜感** | 6-10 个精选，用户会认真看每个 |
| **相关性** | 基于前面的分析结果，推荐真正相关的内容 |
| **可执行** | 每个推荐都有具体建议，不只是"看看" |

### 5.3 灵活性保留

- 用户可以**点击进度条**跳转到任意步骤
- 侧边栏保留快捷入口（高级用户快速访问）
- 每步完成后数据自动保存
- 用户可以随时暂停，下次继续

## 六、数据流

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           数据流向                                       │
└─────────────────────────────────────────────────────────────────────────┘

Step 1: 竞品分析
    │
    ├── 输入：竞品笔记链接
    ├── 处理：小红书数据抓取 + Claude Haiku 分析
    └── 输出：竞品分析结果（存入 session）
          │
          ▼
Step 2: 内容诊断
    │
    ├── 输入：用户笔记链接 + Step 1 的竞品数据
    ├── 处理：小红书数据抓取 + Claude Haiku 分析 + 对比
    └── 输出：诊断结果 + 差距分析（存入 session）
          │
          ▼
Step 3: 可视化对比
    │
    ├── 输入：Step 1 + Step 2 的数据
    ├── 处理：数据聚合 + 图表生成
    └── 输出：可视化对比图表
          │
          ▼
Step 4: 海外创意灵感库
    │
    ├── 输入：前几步的分析结果（用户类别、差距点）
    ├── 处理：YouTube/TikTok API 搜索 + 相关性筛选 + Claude Haiku 解读
    └── 输出：6-10 个精选视频 + 每个的本土化建议
          │
          ▼
Step 5: 完成与提醒
    │
    ├── 输入：所有步骤的数据
    ├── 处理：汇总 + 生成报告
    └── 输出：完整报告 + 下次诊断提醒
```

## 七、技术要点

### 7.1 数据存储

| 阶段 | 存储方式 | 说明 |
|------|---------|------|
| MVP | Session Storage | 临时存储，完成流程后可导出 |
| V2 | SQLite + 用户账号 | 持久化存储，历史记录 |

### 7.2 AI 服务使用

| 功能 | 模型 | 原因 |
|------|------|------|
| 竞品分析 | Claude Sonnet | 需要深度分析 |
| 内容诊断 | Claude Sonnet | 需要深度分析 |
| 灵感解读 | Claude Haiku | 成本敏感，文本分析 |

### 7.3 外部 API

| API | 用途 | 成本 |
|-----|------|------|
| 小红书公开数据 | 竞品/自己笔记分析 | 免费（Playwright） |
| YouTube Data API | 海外灵感推荐 | 免费（10,000 单位/天） |
| TikTok via Apify | 海外灵感推荐 | $49/月 |

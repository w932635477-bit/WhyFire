| 原方向问题 | 新方向优势 |

## 一、产品概述

### 1.1 产品名称

**WhyFire**

- Why = 为什么
- Fire = 火/爆款
- 含义：破解"为什么火"的密码

**Slogan**：打破创作瓶颈，找到你的下一个爆款 / Break the creative block

---

### 1.2 一句话定位

> **小红书博主的海外创意灵感库 — AI 帮你解读热门视频，给你本土化创作建议**

---

### 1.3 核心价值主张

> **"不知道拍什么？看见海外正在火的创意，AI 帮你解读并本土化"**

---

## 二、目标用户

### 2.1 用户定义

| 维度 | 描述 |
|------|------|
| **人群** | 小红书博主（0-1万粉为主） |
| **核心痛点** | "拍着拍着就不知道该拍什么了" |
| **心理状态** | 创意枯竭，缺乏灵感来源 |
| **信息困境** | 只能看到国内同质化内容 |
| **支付能力** | 有限，需要低价/免费方案 |
| **人数规模** | 最大的人群，但被大厂忽视 |

### 2.2 为什么选择这个人群？

| 原因 | 说明 |
|------|------|
| **痛点真实** | "不知道拍什么"比"开头不够好"更严重 |
| **人数多** | 起步博主数量远超成熟博主 |
| **被忽视** | 大厂服务品牌/MCN，起步博主付不起高价 |
| **价值稀缺** | 海外创意库是差异化价值 |
| **可培养** | 从小白开始培养，长期价值高 |

## 三、用户痛点

### 3.1 核心痛点

| 痛点 | 描述 | 严重程度 |
|------|------|---------|
| **创意枯竭** | "拍着拍着就不知道该拍什么了" | ⭐⭐⭐⭐⭐ |
| **信息茧房** | 只能看到国内同质化内容 | ⭐⭐⭐⭐⭐ |
| **缺乏灵感** | 不知道海外流行什么 | ⭐⭐⭐⭐ |
| **本土化困难** | 知道海外好内容，但不知道怎么改 | ⭐⭐⭐ |

### 3.2 核心问题

> "我想创作，但没有灵感；我知道应该学习，但不知道学谁"

### 3.3 解决方案

```
用户现状                    WhyFire 解决方案                用户收益
────────                    ────────────────                ────────
只看国内同质化内容    →     聚合海外热门创意内容      →     发现新灵感
不知道什么在流行      →     TikTok/IG/YouTube 趋势   →     紧跟全球热点
不懂如何借鉴         →     AI 解读创意要点         →     可执行建议
照搬可能水土不服     →     本土化建议生成          →     适配小红书
```

## 五、技术方案

### 5.1 技术架构 v2.0

```
┌─────────────────────────────────────────────────────────────┐
│  数据源层（混合方案）                                        │
│  ├── YouTube Data API v3（主数据源，70%）                    │
│  │   → Shorts trending、分类热门                            │
│  │   → 官方 API，免费稳定                                   │
│  │                                                          │
│  └── TikTok via Apify（辅助数据源，30%）                     │
│      → Trending videos、hashtag 热门                        │
│      → 商业服务，稳定                                       │
├─────────────────────────────────────────────────────────────┤
│  AI 分析层（文本分析，低成本）                               │
│  ├── 输入：标题 + 描述 + 标签 + 互动数据                     │
│  ├── 模型：Claude Haiku（$0.25/1M tokens）                  │
│  └── 输出：创意要点 + 本土化建议 + 推荐标题/标签             │
├─────────────────────────────────────────────────────────────┤
│  展示层                                                      │
│  └── Next.js 15 + React 18 + Tailwind CSS                   │
│      → 瀑布流卡片 + AI 解读面板 + 筛选器                     │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 核心依赖

| 依赖 | 用途 | 成本 |
|------|------|------|
| **YouTube Data API v3** | YouTube 热门数据 | 免费（10,000单位/天） |
| **Apify TikTok Scraper** | TikTok 热门数据 | $49/月（可按需） |
| **Claude Haiku** | 文本分析 | ~$5-10/月 |
| **Next.js 15** | 全栈框架 | 开源 |

**月成本估算：约 ¥400/月**

### 5.3 为什么选择文本分析

| 维度 | 视频分析 | 文本分析（采用） |
|------|---------|-----------------|
| 成本 | $3/1M tokens | $0.25/1M tokens |
| 延迟 | 数秒 | 毫秒级 |
| 信息量 | 视觉细节 | 元数据足够 |
| 复杂度 | 需要 FFmpeg | 纯 API |

**结论**：对于"创意灵感"场景，标题/描述/标签已包含核心创意信息

```
用户来浏览 → 发现海外热门 → 看AI解读 → 获得本土化建议 → 创作新内容
     ↑                                              ↓

```

### 6.2 为什么用户会回来？

| 原因 | 说明 |
|------|------|
| **内容持续更新** | 海外热门每天更新，每天有新灵感 |

## 八、差异化定位

### 8.1 vs 商业竞品

| 维度 | 千瓜/新红 | WhyFire |
|------|----------|---------|
| 目标用户 | 品牌/MCN | 0-1万粉起步博主 |
| 价格 | 年费数千~数万 | Freemium（免费+¥49-99/月） |
| 核心价值 | 数据报表 | AI 解读 + 本土化建议 |
| 数据来源 | 国内平台 | 海外平台（差异化） |
| 学习成本 | 需要培训 | 5分钟上手 |

### 8.2 核心竞争力

```
1. 数据源差异化：海外创意 vs 国内同质化
2. 用户定位：被大厂忽视的起步博主
3. 价格策略：他们付得起的低价
4. 核心价值：不是数据，是"可执行建议"
5. 使用体验：零门槛，打开即用
```

## 九、信任建立策略

### 9.1 信任金字塔

```
第一层：零门槛体验
- 完全免费浏览，无需注册
- 打开即看到海外热门
- 先给价值，再谈付费

第二层：真实案例展示
- 展示 AI 解读的实际效果
- "这个视频为什么火"
- 用真实数据证明价值

第三层：透明度
- 告诉用户 AI 如何分析
- 展示分析依据
- 让用户能验证合理性

第四层：隐私承诺
- 不存储用户个人信息
- 收藏数据加密存储
```
